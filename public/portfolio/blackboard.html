Blackboard Learn

Blackboard Learn is one of the most widely used Learning Management Systems in the world. A new theme and responsive design help bring this classic LMS into the modern web.

The Learn 2016 theme for Blackboard Learn is a modern version of the user interface that incorporates elements from the new Blackboard design language including color, fonts, and spacing for greater consistency between Blackboard Learn, Collaborate, and the Blackboard app. This optional theme is a first step in improving the experience for mobile users. Instructors and students can access their course materials whether they're at a desk or on the go!

Along with modernizing the look and feel of Learn, UI regressions tests using backstop JS, Docker, Kubernetes, and Jenkins were developed. This ensures a consistent bug free UI for Learn as new development continues.
