Jason Eggert UX Designer/Developer

Batesville Casket

Using technology better assist families and funeral professionals plan final arrangements for loved ones. Created web application and product management system to allow funeral professionals to curate selections and content to walk customers through the final arrangements process.

Batesville Casket wanted to create a tool to improve the planning of final arrangements. 2 solutions, that work together were developed. First a customer facing web application that contains curated collections and information to help families through the process.

Then a back-end admin application was developed. Through this application funeral home owners and directors could view, group, price, products and process information.
